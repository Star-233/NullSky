**标准长相**：
$$y'' + py' + qy = f(x)$$
（右边不等于 0）
**总通解结构**：
$$\boxed{y = Y + y^*}$$
- $Y$ 也就是上面**方程对应的齐次微分方程**算出来的齐次通解。
- $y^*$ 是需要根据 $f(x)$ 设出来的**非齐次特解**。
    

### 1. 核心设代公式（待定系数法）
根据方程右边 $f(x)$ 的长相，分为两大类型：
#### 类型 Ⅰ：$f(x) = e^{\lambda x} P_m(x)$ （指数 $\times$ 多项式）
- **特解设法**：
    
    $$\boxed{y^* = x^k R_m(x) e^{\lambda x}}$$
    
- **参数怎么填**：
    - **$R_m(x)$**：和 $P_m(x)$ 同次数的完整多项式（系数用 $A, B, C \dots$ 待定）。
    - **$k$ 的取值**：看 $\lambda$ 是不是特征方程的根。
        - $\lambda$ 不是特征根 $\implies k = 0$
        - $\lambda$ 是单根（等于 $r_1$ 或 $r_2$） $\implies k = 1$
        - $\lambda$ 是重根（等于 $r_1 = r_2$） $\implies k = 2$
            

#### 类型 Ⅱ：$f(x) = e^{\lambda x} [P_l(x)\cos \omega x + Q_n(x)\sin \omega x]$ （含三角函数）
- **特解设法**：
    $$\boxed{y^* = x^k e^{\lambda x} [R_m^{(1)}(x)\cos \omega x + R_m^{(2)}(x)\sin \omega x]}$$
- **参数怎么填**：
    - **$m$ 的次数**：取 $l$ 和 $n$ 中最高的那一个（$m = \max\{l, n\}$）。
    - **$R_m^{(1)}, R_m^{(2)}$**：设两个 $m$ 次的完整多项式。
    - **$k$ 的取值**：看 $\lambda \pm \omega i$ 是不是特征方程的根。
        - $\lambda \pm \omega i$ 不是特征根 $\implies k = 0$
        - $\lambda \pm \omega i$ 是特征根 $\implies k = 1$

---

> [!example] 指数 $\times$ 多项式
> 求解微分方程：
> $$y'' - 3y' + 2y = x e^x$$
### 求齐次通解
先求对应齐次方程 $y'' - 3y' + 2y = 0$ 的通解，其特征方程为
$$r^2 - 3r + 2 = 0 \implies (r-1)(r-2) = 0 \implies r_1 = 1, \; r_2 = 2$$
查表格得到其通解
$$Y = C_1 e^x + C_2 e^{2x}$$

### 设特解
- **$\lambda$**：$e^{1x}$ 说明 **$\lambda = 1$**。
- $k$：我们的特征根 $r_1=\lambda=1$ 所以 $k=1$
- $m$：在这里 $P_m(x)=x$ ，只有 1 次，所以 $m=1$
- $R_m(x)$：既然 $m=1$ ，所以我们设 $R_m(x) = Ax + B$

套进公式里得
$$y^* = x^1 (Ax + B)e^x = (Ax^2 + Bx)e^x$$
### 解特解系数
我们现在仅仅是设出了特解，还需要将其代回原方程解系数
把 $y^*$ 求导两次（注意用乘积求导法则）：
- $y^{*\prime} = [Ax^2 + (2A+B)x + B]e^x$
- $y^{*\prime\prime} = [Ax^2 + (4A+B)x + 2A+2B]e^x$
把它们狠狠地代入原方程 $y'' - 3y' + 2y = x e^x$，消去 $e^x$ 并化简合并同类项后得到：
$$-2Ax + (2A - B) = x$$

对比两边 $x$ 的系数和常数项：
- $-2A = 1 \implies A = -\frac{1}{2}$
- $2A - B = 0 \implies B = 2A = -1$

所以，特解为：
$$y^* = \left(-\frac{1}{2}x^2 - x\right)e^x$$
### 合并齐次通解与特解
通解为
$$y = Y + y^* = C_1 e^x + C_2 e^{2x} - \left(\frac{1}{2}x^2 + x\right)e^x$$

> [!example] 含三角函数
> 求解微分方程：
> $$y'' + y = \sin x$$

### 求齐次通解
对应齐次方程的特征方程
$$r^2 + 1 = 0 \implies r^2 = -1 \implies r_{1,2} = \pm i$$
这属于共轭复根（实部 $\alpha = 0$，虚部 $\beta = 1$）。查表可得齐次通解为：
$$Y = e^{0x}(C_1 \cos 1x + C_2 \sin 1x) = C_1 \cos x + C_2 \sin x$$

### 设特解
- $\lambda$：没有 $e^{x}$ 说明 $\lambda = 0$
- $\omega$：$\sin 1x$ 说明 $\omega = 1$
- $k$：$\lambda + \omega i$ 为 $i$ ，是之前特征根的解，所以 $k=1$
- $m$：题目里没有 $\cos x$ ，有 $\sin x$ ，其对应的 $m=0$ ，这就是 $m$ 的取值了

然后我们就可以设出特解
$$y^* = x^1 e^{0x} [A \cos x + B \sin x] = x (A \cos x + B \sin x)$$

### 算特解的系数
把 $y^*$ 求导两次：
- $y^{*\prime} = (A + Bx)\cos x + (B - Ax)\sin x$
- $y^{*\prime\prime} = (2B - Ax)\cos x - (2A + Bx)\sin x$
    

代回原方程 $y'' + y = \sin x$（你会发现带有 $x$ 的项会全部神奇地消掉）：
$$2B \cos x - 2A \sin x = \sin x$$

对比两边 $\cos x$ 和 $\sin x$ 的系数：
- $2B = 0 \implies B = 0$
- $-2A = 1 \implies A = -\frac{1}{2}$
所以，特解为：
$$y^* = -\frac{1}{2}x \cos x$$

### 合并齐次通解与特解
$$y = Y + y^* = C_1 \cos x + C_2 \sin x - \frac{1}{2}x \cos x$$
