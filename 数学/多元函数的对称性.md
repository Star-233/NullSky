> [!question]
>  设 $I_1 = \iint_{D_1} (x^2 + y^2)^3 \, d\sigma$，其中 $D_1 = \{(x,y) \mid -1 \le x \le 1, -2 \le y \le 2\}$；又 $I_2 = \iint_{D_2} (x^2 + y^2)^3 \, d\sigma$，其中 $D_2 = \{(x,y) \mid 0 \le x \le 1, 0 \le y \le 2\}$。试利用二重积分的几何意义说明 $I_1$ 与 $I_2$ 之间的关系。

- 如果一个函数要有对称性，他的定义域也要对称
在这里，很容易看出来 $D_1$ 是关于 $(0,0)$ 对称的

|**测试动作**|**数学表达**|**结论**|
|---|---|---|
|**把 $x$ 换成 $-x$**|$f(-x, y) = -f(x, y)$|对 $x$ 是**奇函数**|
||$f(-x, y) = f(x, y)$|对 $x$ 是**偶函数**|
|**把 $y$ 换成 $-y$**|$f(x, -y) = -f(x, y)$|对 $y$ 是**奇函数**|
||$f(x, -y) = f(x, y)$|对 $y$ 是**偶函数**|
|**把 $x, y$ 互换**|$f(y, x) = f(x, y)$|关于 $y=x$ **轮换对称**|
在这里
- **关于 $y$ 轴对称性：** $f(-x, y) = ((-x)^2 + y^2)^3 = (x^2 + y^2)^3 = f(x, y)$。说明函数关于 $y$ 轴是**偶函数**。
- **关于 $x$ 轴对称性：** $f(x, -y) = (x^2 + (-y)^2)^3 = (x^2 + y^2)^3 = f(x, y)$。说明函数关于 $x$ 轴也是**偶函数**。

可得
$$I_1 = 4I_2$$
