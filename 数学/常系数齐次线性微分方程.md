**标准长相**：
$$y'' + py' + qy = 0$$
（右边等于 0）

### 1. 核心查表公式
拿到方程，直接默写特征方程：
$$r^2 + pr + q = 0$$

，解出特征根 $r_1, r_2$。然后对照下表直接写答案：

| **特征根 r1​,r2​ 的情况**                       | **判别式 Δ**    | **对应的齐次通解 Y 公式**                                                   |
| ----------------------------------------- | ------------ | ------------------------------------------------------------------ |
| **两个不相等的实根** $r_1 \neq r_2$               | $\Delta > 0$ | $$\boxed{Y = C_1 e^{r_1 x} + C_2 e^{r_2 x}}$$                      |
| **两个相等的实根** $r_1 = r_2 = r$               | $\Delta = 0$ | $$\boxed{Y = (C_1 + C_2 x)e^{r x}}$$                               |
| **一对共轭复根** $r_{1,2} = \alpha \pm \beta i$ | $\Delta < 0$ | $$\boxed{Y = e^{\alpha x} (C_1 \cos \beta x + C_2 \sin \beta x)}$$ |
或者看这个推广版本也行：

|**特征根的类型**|**根的具体情况**|**通解中对应的组合项**|**考试暗号（Pattern）**|
|---|---|---|---|
|**单实根**|$r$ （只出现 1 次）|$$C e^{rx}$$|最基础的指数项|
|**$k$ 重实根**|$r$ （重复出现了 $k$ 次）|$$(C_1 + C_2 x + C_3 x^2 + \dots + C_k x^{k-1})e^{rx}$$|出现几次，括号里就用 $x$ 的几次幂排队|
|**单对共轭复根**|$\alpha \pm \beta i$ （只出现 1 次）|$$e^{\alpha x}(C_1 \cos \beta x + C_2 \sin \beta x)$$|实部在指数，虚部在三角|
|**$k$ 重共轭复根**|$\alpha \pm \beta i$ （重复出现了 $k$ 次）|$$e^{\alpha x} \left[ (C_1 + \dots + C_k x^{k-1})\cos \beta x + (D_1 + \dots + D_k x^{k-1})\sin \beta x \right]$$|三角函数前面的系数，也变成了 $x$ 的排队多项式|
### 2. 简要操作指南
- **Step 1**：把 $y''$ 换成 $r^2$，$y'$ 换成 $r$，$y$ 换成 $1$，得到一个普通的一元二次方程。
- **Step 2**：用十字相乘法或求根公式把 $r_1, r_2$ 算出来。
- **Step 3**：看它们属于上面哪一栏，直接把 $r$（或 $\alpha$ 和 $\beta$）代入方框公式。$C_1, C_2$ 原封不动保留。

---


> [!example] 两个不相等的实根
> 求解微分方程：
> $$y'' - y' - 6y = 0$$

写出特征方程
$$r^2 - r - 6 = 0$$
$$(r-3)(r+2) = 0$$
解出根
$$r_1 = 3, \quad r_2 = -2$$
查表，这对应**两个不相等的实根**，则
$$y = C_1 e^{3x} + C_2 e^{-2x}$$


> [!example] 一对共轭复根
> 求解微分方程
> $$y'' - 4y' + 13y = 0$$

写出特征方程
$$r^2 - 4r + 13 = 0$$
用求根公式 $r = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$ 求解
$$r = \frac{-(-4) \pm \sqrt{(-4)^2 - 4 \times 1 \times 13}}{2} = \frac{4 \pm \sqrt{16 - 52}}{2} = \frac{4 \pm \sqrt{-36}}{2}$$
$$r = \frac{4 \pm 6i}{2} = 2 \pm 3i$$
> **实部 $\alpha = 2$**，**虚部 $\beta = 3$**（注意：$\beta$ 永远取正数，不带 $i$）。

查表得
$$y = e^{2x} (C_1 \cos 3x + C_2 \sin 3x)$$
