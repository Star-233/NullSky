
### 1. 三角函数类

这是最基础、最常用的类别。

| 函数 | 等价无穷小 ($x \to 0$) |
| --- | --- |
| $\sin x$ | $\sim x$ |
| $\tan x$ | $\sim x$ |
| $\arcsin x$ | $\sim x$ |
| $\arctan x$ | $\sim x$ |
| **$1 - \cos x$** | $\sim \frac{1}{2}x^2$ |

---

### 2. 指数与对数类

处理包含 $e$、$\ln$ 等复合函数极限时的核心。

| 函数 | 等价无穷小 ($x \to 0$) |
| --- | --- |
| $e^x - 1$ | $\sim x$ |
| $a^x - 1$ | $\sim x \ln a$ |
| $\ln(1+x)$ | $\sim x$ |
| $\log_a(1+x)$ | $\sim \frac{x}{\ln a}$ |

---

### 3. 幂函数与根式类

处理开方或复杂幂次时的“降维打击”。

| 函数 | 等价无穷小 ($x \to 0$) |
| --- | --- |
| $(1+x)^\alpha - 1$ | $\sim \alpha x$ |
| $\sqrt{1+x} - 1$ | $\sim \frac{1}{2}x$ |
| $\sqrt[n]{1+x} - 1$ | $\sim \frac{1}{n}x$ |

---
#### 常见进阶组合

* $x - \sin x \sim \frac{1}{6}x^3$
* $\tan x - x \sim \frac{1}{3}x^3$
* $x - \ln(1+x) \sim \frac{1}{2}x^2$
