[冶炼金属](https://www.lanqiao.cn/problems/3510/learning/)
题目的意思给出一系列 $B < A$
然后已知 $\lfloor \frac{A}{B} \rfloor = V$ ，求 $V$ 的可能值。

和[[LCM  公式化简]]这道题目类似，这个向下取整我们没法直接动，我的意思是我们没法对他进行移项之类的操作。所以我们像遇到括号就想拆括号一样，根据定义把他拆开。

向下取整的定义是，$\lfloor x \rfloor = n$ 等价于不等式组：
$$n \le x < n + 1$$
那么
$$
B \le \frac{A}{V} < B+1
$$
容易得到
$$V \le \frac{A}{B}$$
$$V > \frac{A}{B + 1}$$
那么

$$V_{max} = \lfloor \frac{A}{B} \rfloor$$
$$V_{min} = \lfloor \frac{A}{B + 1} \rfloor + 1$$
