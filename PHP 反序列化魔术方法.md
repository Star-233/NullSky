| 方法              | 触发时机                   |
| --------------- | ---------------------- |
| `__destruct()`  | 对象销毁时                  |
| `__wakeup()`    | 反序列化后立即执行              |
| `__toString()`  | 对象转字符串时                |
| `__call()`      | 调用不存在的实例方法             |
| `__invoke()`    | 对象作为函数调用               |
| `__construct()` | 创建对象时触发，注意反序列化的时候并不会触发 |
