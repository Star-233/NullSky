
| **术语 (英/中)**            | **通俗解释**                      |
| ----------------------- | ----------------------------- |
| **Agent / 智能体**         | 玩家，也就是正在学跳舞的那个 AI。            |
| **Environment / 环境**    | 游戏世界，负责告诉你物理规则。               |
| **State / 状态 (obs)**    | 机器人现在的“处境”。                   |
| **Action / 动作**         | 机器人对处境做出的“反应”。                |
| **Reward / 奖励 ($R$)**   | 甜头或苦头。分数越高，AI 越觉得这招对。         |
| **Policy / 策略 ($\pi$)** | AI 的“条件反射”手册（看到 A 状态就做 B 动作）。 |
| **Step / 步**            | 游戏里的“一帧”。                     |
| **Episode / 回合**        | 游戏里的“一局”。                     |
