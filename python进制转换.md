
## 其他进制转10进制
```python
# 二进制 -> 十进制
print(int('1010', 2))   # 10

# 八进制 -> 十进制
print(int('12', 8))     # 10

# 十六进制 -> 十进制
print(int('a', 16))     # 10
```

## 10进制转其他进制
| 目标进制 | 函数      | 示例        | 输出         |
| ---- | ------- | --------- | ---------- |
| 二进制  | `bin()` | `bin(10)` | `'0b1010'` |
| 八进制  | `oct()` | `oct(10)` | `'0o12'`   |
| 十六进制 | `hex()` | `hex(10)` | `'0xa'`    |
