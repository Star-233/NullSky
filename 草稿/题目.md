# 多元函数偏导数计算练习题

## 一、选择题

1. 设函数 $f(x, y) = x^2 + 3xy + y^2$，则 $\frac{\partial f}{\partial x}(1, 2)$ 等于：
   A. 5
   B. 7
   C. 8
   D. 10


2. 设 $z = x^2 + y^2 + 2xy$，则 $\frac{\partial^2 z}{\partial x \partial y}$ 等于（  ）
   A. 0
   B. 1
   C. 2
   D. 4

3. 设函数 $z = x^2y + \sin(xy)$，则 $\frac{\partial^2 z}{\partial x \partial y}$ 在点 $(1, \pi)$ 处的值为（ ）
   A. $2 + \cos\pi$
   B. $2\pi + \cos\pi$
   C. $2\pi - \cos\pi$
   D. $2 - \cos\pi$

4. 设 $z = f(u, v)$，其中 $u = x + y$，$v = xy$，且 $f$ 具有连续偏导数。则 $\frac{\partial z}{\partial x}$ 等于：
   A. $f_u + yf_v$
   B. $f_u + xf_v$
   C. $xf_u + yf_v$
   D. $yf_u + xf_v$

5. 设方程 $e^z = xyz$ 确定了隐函数 $z = f(x, y)$，则 $\frac{\partial z}{\partial x}$ 等于：
   A. $\frac{yz}{e^z - xy}$
   B. $\frac{yz}{e^z + xy}$
   C. $\frac{-yz}{e^z - xy}$
   D. $\frac{-yz}{e^z + xy}$

6. 设函数 $f(x, y) = x^y$（$x > 0$），则 $\frac{\partial f}{\partial x}$ 等于（ ）
   A. $yx^{y-1}$
   B. $x^y \ln x$
   C. $y^x \ln y$
   D. $xy^{y-1}$

7. 设 $z = f(u, v)$，其中 $u = x + y$，$v = xy$，$f$ 具有二阶连续偏导数，则 $\frac{\partial^2 z}{\partial x^2}$ 等于（ ）
   A. $f_{uu} + 2y f_{uv} + y^2 f_{vv} + f_v$
   B. $f_{uu} + 2y f_{uv} + y^2 f_{vv}$
   C. $f_{uu} + 2x f_{uv} + x^2 f_{vv} + f_v$
   D. $f_{uu} + 2xy f_{uv} + x^2 y^2 f_{vv} + f_v$

## 二、填空题

8. 设函数 $g(x, y) = e^{xy} + \sin(x + y)$，则 $\frac{\partial g}{\partial y} = \underline{\qquad}$。

9. 设 $z = x^3 y^2$，则 $\frac{\partial^2 z}{\partial x^2} =$ ______，$\frac{\partial^2 z}{\partial y^2} =$ ______

10. 设 $f(x, y, z) = e^{xyz}$，则：
    1. $f_x = $ ________
    2. $f_{xy} = $ ________
    3. $f_{xyz} = $ ________

11. 设 $z = e^{u}\sin v$，其中 $u = xy$，$v = x + y$，则 $\frac{\partial z}{\partial x} = \underline{\qquad}$。

12. 设方程 $x^2 + y^2 + z^2 = 1$ 确定了隐函数 $z = f(x, y)$，则 $\frac{\partial z}{\partial y} = \underline{\qquad}$。

13. 设函数 $f(x, y) = \ln(x^2 + y^2)$，则：
    1. $\frac{\partial f}{\partial x} =$ ________
    2. $\frac{\partial^2 f}{\partial x \partial y} =$ ________

14. 设方程 $xyz + x^2 + y^2 + z^2 = 2$ 确定隐函数 $z = f(x, y)$，则 $\frac{\partial^2 z}{\partial x^2}$ 在点 $(1, 0, -1)$ 处的值为 ______。

## 三、判断题

15. 若函数 $f(x, y)$ 在点 $(a, b)$ 处对 $x$ 的偏导数存在，则 $f(x, y)$ 在点 $(a, b)$ 处一定连续。（ ）

16. 如果函数 $z = f(x, y)$ 的二阶偏导数都在某区域内连续，那么在该区域内必有 $\frac{\partial^2 z}{\partial x \partial y} = \frac{\partial^2 z}{\partial y \partial x}$（  ）

17. 若二元函数 $z = f(x, y)$ 的二阶混合偏导数 $f_{xy}$ 和 $f_{yx}$ 都存在，则一定有 $f_{xy} = f_{yx}$。（ ）

18. 若方程 $F(x, y, z) = 0$ 在点 $(x_0, y_0, z_0)$ 的某邻域内有连续偏导数，且 $F(x_0, y_0, z_0) = 0$，$F_z(x_0, y_0, z_0) \neq 0$，则该方程在点 $(x_0, y_0, z_0)$ 的某邻域内唯一确定了隐函数 $z = f(x, y)$，且 $f(x_0, y_0) = z_0$。（ ）
