形如
$$
{\frac {d y}{d x}}+P(x)\,y=Q(x) \tag{1}
$$
的微分方程称之为一阶线性微分方程

## 齐次情况的通解
当 $Q(x)=0$ 的时候，方程 $(1)$ 就是齐次的。他的通解为
$$
y = C \mathrm{e}^{-\int P(x) \mathrm{d} x}
$$

## 非齐次情况的通解
当 $Q(x) \ne 0$ 的时候，方程 $(1)$ 就不是齐次的。此时他的通解为
$$
y = { \mathrm { e } } ^ { - \int P ( x ) \,\mathrm { d } x } \left[ \int Q ( x ) \, { \mathrm { e } } ^ { \int P ( x ) \,\mathrm { d } x } \,\mathrm { d } x + C \right]
$$
