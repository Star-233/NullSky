## 二分查找
### 整数二分
#### 查找第一个 $\ge x$ 的元素位置
```cpp
int find_first_ge(const vector<int>& a, int x) {
    int l = 0, r = (int)a.size() - 1;
    while (l <= r) {
        int mid = l + (r - l) / 2;
        if (a[mid] >= x) r = mid - 1; // 答案在 mid 或左边
        else l = mid + 1;
    }
    return l; // 边界检查：使用前需判断 l < n
}
```
#### 查找第一个 $> x$ 的元素位置
```cpp
int find_first_gt(const vector<int>& a, int x) {
```

### 实数二分
### 固定次数控制精度
```cpp
double real_bsearch_iterations(double l, double r) {
    // 循环 100 次，此时区间范围缩小了 2^(-100)，精度远远超过了 double 的极限
    for (int i = 0; i < 100; i++) {
        double mid = l + (r - l) / 2.0;
        
        if (check(mid)) {
            r = mid; // 或者是 l = mid，根据题目要求的单调性决定
        } else {
            l = mid;
        }
    }
    return l; // 此时 l 和 r 几乎完全重合，返回谁都可以
}
```
### 设置精度上限来控制精度
```cpp
double real_bsearch_eps(double l, double r) {
    // eps 应当比题目要求的精度再小 2 到 3 个数量级
    // 例如题目要求保留 6 位小数（1e-6），eps 取 1e-8 或 1e-9
    const double eps = 1e-9; 
    
    while (r - l > eps) {
        double mid = l + (r - l) / 2.0;
        
        if (check(mid)) {
            r = mid;
        } else {
            l = mid;
        }
    }
    return l;
}
```

## 前缀与差分

### 一维前缀
```cpp
// 假设数组大小为 n，数据从 1 到 n 存储
vector<int> a(n + 1);
vector<int> S(n + 1, 0);

// 1. 预处理前缀和数组
for (int i = 1; i <= n; i++) {
    S[i] = S[i - 1] + a[i];
}

// 2. O(1) 查询区间 [l, r] 的和
int query(int l, int r) {
    return S[r] - S[l - 1];
}
```
### 一维差分
```cpp
vector<int> b(n + 2, 0); // 差分数组，注意多开一位防止 r + 1 越界

// 1. O(1) 区间修改：给 [l, r] 区间加上 c
void insert(int l, int r, int c) {
    b[l] += c;
    b[r + 1] -= c;
}

// 2. O(N) 恢复原数组
for (int i = 1; i <= n; i++) {
    a[i] = a[i - 1] + b[i]; // 此时 a[i] 就是修改后的最终值
}
```
### 二维前缀
```cpp
int a[N][M], S[N][M]; // 注意也是要多开空间的

// 1. 预处理二维前缀和
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= m; j++) {
        S[i][j] = S[i - 1][j] + S[i][j - 1] - S[i - 1][j - 1] + a[i][j];
    }
}

// 2. O(1) 查询子矩阵和
int query_2d(int x1, int y1, int x2, int y2) {
    return S[x2][y2] - S[x1 - 1][y2] - S[x2][y1 - 1] + S[x1 - 1][y1 - 1];
}
```
### 二维差分
```cpp
int b[N][M]; // 差分矩阵,需要多开空间

// 1. O(1) 修改子矩阵
void insert_2d(int x1, int y1, int x2, int y2, int c) {
    b[x1][y1] += c;
    b[x2 + 1][y1] -= c;
    b[x1][y2 + 1] -= c;
    b[x2 + 1][y2 + 1] += c;
}

// 2. O(N×M) 恢复原矩阵（直接对差分矩阵求二维前缀和）
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= m; j++) {
        b[i][j] += b[i - 1][j] + b[i][j - 1] - b[i - 1][j - 1];
        // 此时 b[i][j] 就是该位置最终被修改后的值
    }
}
```
## 滑动窗口
### 滑动窗口最大值
```cpp
vector<int> maxSlidingWindow(vector<int>& a, int k) {
    vector<int> ans;
    deque<int> q; // 存储下标
    
    for (int i = 0; i < a.size(); i++) {
        // 1. 查过期
        if (!q.empty() && q.front() < i - k + 1) {
            q.pop_front();
        }
        // 2. 踢无用（以求最大值为例：新来的大，旧的就从后面弹出）
        while (!q.empty() && a[q.back()] <= a[i]) {
            q.pop_back();
        }
        // 3. 进新人
        q.push_back(i);
        // 4. 取答案
        if (i >= k - 1) {
            ans.push_back(a[q.front()]);
        }
    }
    return ans;
}
```

## 数学
### 质数筛法
#### 欧拉筛（线性筛）
- **时间复杂度：** $O(N)$，每个合数严格只被它的**最小质因数**筛掉一次。
- **空间复杂度：** $O(N)$
```cpp
const int N = 1000010; // 根据题目范围调整大小，记得多开几个
int primes[N], cnt;    // primes 存放所有质数, cnt 是质数个数
bool st[N];            // st[i] 标记 i 是否被筛过（true 表示不是质数/已被筛）

void get_primes(int n) {
    st[0] = st[1] = true; // 0 和 1 不是质数
    for (int i = 2; i <= n; i++) {
        if (!st[i]) primes[cnt++] = i; // 如果没被筛过，说明是质数
        
        // 核心：用当前已有的质数去筛掉合数
        for (int j = 0; primes[j] <= n / i; j++) {
            st[primes[j] * i] = true; // 筛掉合数
            
            // 🌟 终极核心：线性筛的精髓
            // 当 i 能整除 primes[j] 时，primes[j] 一定是 i 的最小质因数。
            // 那么 primes[j] 也一定是 (primes[j] * i) 的最小质因数。
            // 如果不 break，后续的合数就会被更大的质因数筛掉，导致重复计算，失去 O(N) 效率。
            if (i % primes[j] == 0) break; 
        }
    }
}
```
#### 埃氏筛
**时间复杂度：** $O(N \log \log N)$，速度略慢于线性筛，但在 $10^6$ 级别以内几乎无差别。
```cpp
const int N = 1000010; // ⚠️ 这里的 N 必须大于题目要求的最大范围 n

int primes[N]; // 存放质数的数组
int cnt;       // 记录质数的个数。全局变量默认初始化为 0
bool st[N];    // 标记数组。全局变量默认全为 false（false 表示是质数，true 表示被筛掉了）

void get_primes_era(int n) {
    st[0] = st[1] = true;
    for (int i = 2; i <= n; i++) {
        if (!st[i]) {
            primes[cnt++] = i;
            // 优化：从 i * i 开始筛，防止重复
            // ⚠️ 注意：如果 n 很大，i * i 可能会爆 int，需要转成 long long
            for (int j = i * i; j <= n; j += i) st[j] = true;
        }
    }
}
```

### 快速幂
快速幂用于在 $O(\log b)$ 的时间内计算 $a^b \pmod p$ 的值。它的原理是利用指数的**二进制拆分**。
```cpp
using ll = long long;

// 计算 (a^b) % p
ll qmi(ll a, ll b, ll p) {
    ll res = 1 % p; // 考虑 p = 1 的极端情况
    while (b) {
        if (b & 1) res = res * a % p; // 如果当前二进制位是 1，累乘到答案中
        a = a * a % p;                // 权值翻倍：a -> a^2 -> a^4 -> a^8...
        b >>= 1;                      // 指数右移一位
    }
    return res;
}
```
