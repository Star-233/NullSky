## 🔗 原型链是什么？

JavaScript 中每个对象都有一个隐藏的 `[[Prototype]]`（可通过 `__proto__` 或 `Object.getPrototypeOf()` 访问）。当访问一个对象的属性时，**如果该对象自身没有这个属性，JS 引擎会沿着原型链向上查找，直到找到或到达 `null`。**

```javascript
let obj = { a: 1 };
console.log(obj.toString); // 来自 Object.prototype
```

这种继承机制使得所有普通对象都共享 `Object.prototype` 上的方法（如 `toString`、`hasOwnProperty`）。

不一定所有的原型链都到达 `Object.prototype`

| 对象示例                              | 原型链 (从当前对象到 `null`)                                                                    |
| :-------------------------------- | :------------------------------------------------------------------------------------- |
| 对象字面量 `{}` 或 `new Object()`       | `当前对象` → `Object.prototype` → `null`                                                   |
| 数组 `[]` 或 `new Array()`           | `当前对象` → `Array.prototype` → `Object.prototype` → `null`                               |
| 函数 `function(){}` 或 `() => {}`    | `当前对象` → `Function.prototype` → `Object.prototype` → `null`                            |
| 正则表达式 `/./` 或 `new RegExp('.')`   | `当前对象` → `RegExp.prototype` → `Object.prototype` → `null`                              |
| 日期 `new Date()`                   | `当前对象` → `Date.prototype` → `Object.prototype` → `null`                                |
| 错误对象 `new Error()`                | `当前对象` → `Error.prototype` → `Object.prototype` → `null`                               |
| 自定义构造函数实例 `new Person('Alice')`   | `当前实例` → `Person.prototype` → `Object.prototype` → `null`                              |
| 使用 `Object.create(proto)` 创建      | `当前对象` → `proto` → (继续向上) → … → `null`                                                 |
| 使用 `Object.create(null)` 创建       | `当前对象` → `null` (没有原型链)                                                                |
| 基本类型包装对象 (通过 `new String('foo')`) | `当前对象` → `String.prototype` → `Object.prototype` → `null` <br> (同理 `Number`、`Boolean`) |
| `Object.prototype` 自身             | `Object.prototype` → `null`                                                            |
| `Function.prototype`              | `Function.prototype` → `Object.prototype` → `null`                                     |
| `Array.prototype`                 | `Array.prototype` → `Object.prototype` → `null`                                        |

## 🧪 什么是“污染”？

**原型链污染**是指攻击者能够修改 `Object.prototype`（或其他关键原型）的属性，导致所有继承自该原型的对象都“获得”了这个属性。

```javascript
// 正常情况
let obj1 = {};
let obj2 = {};
console.log(obj1.polluted); // undefined

// 污染原型
Object.prototype.polluted = 'yes';

console.log(obj1.polluted); // 'yes'  (obj1 自身没有，但原型上有)
console.log(obj2.polluted); // 'yes'
```

一旦 `Object.prototype` 被注入恶意属性，整个应用程序几乎所有对象都会受到影响。

## 🎯 典型污染入口

常见于以下不安全的操作：

```js
// 递归合并，没有专门处理 __proto__
function merge(target, source) {
	// 遍历源对象的所有可枚举属性（包括继承来的属性，例如 __proto__）
    for (let key in source) {
	    // 判断源对象的当前属性值是否为对象（且不为 null），是值就赋值
	    // 注意：typeof null === 'object'，所以需要额外排除 null
        if (typeof source[key] === 'object' && source[key] !== null) {
            if (!target[key]) target[key] = {};
            merge(target[key], source[key]);
        } else {
            target[key] = source[key];
        }
    }
    return target;
}
```

最常见的手工 payload：

```json
{
    "__proto__": {
        "isAdmin": true
    }
}
```

如果在服务端将这个 JSON 合并到配置对象中，就会污染原型。

## 💥 危害示例

### 1. 权限绕过
```javascript
function login(user) {
    if (user.isAdmin) grantAccess();
}
// 攻击者污染 Object.prototype.isAdmin = true
// 之后任何 user 对象（即使没有 isAdmin 属性）都会被认为是 admin。
```

### 2. 导致 DoS / 异常行为
```javascript
Object.prototype.toString = null;
// 此后任何隐式调用 toString 的地方都会报错。
```

### 3. RCE（搭配其他 gadget）
如果框架的某个方法会执行原型链上的函数/属性，例如：
```javascript
// 污染 Object.prototype.shell = "require('child_process').exec('calc')"
// 后续某处 eval(obj.shell) 就能执行代码。
```

## 🛡️ 如何防御？

1. **冻结原型**（早期防御）：
   ```javascript
   Object.freeze(Object.prototype);
   ```
2. **使用 `Object.create(null)`** 创建无原型对象存储用户数据。
3. **避免使用不安全的递归合并**，改用 `Object.assign`（不会遍历原型链，但本身只能合并第一层）。
4. **对输入进行过滤**：拒绝包含 `__proto__`、`constructor`、`prototype` 的键。
5. **升级依赖**：`lodash.merge` 等库已在后续版本中修复原型链污染（通过忽略 `__proto__` 属性）。
6. **使用 Map 替代普通对象**存储键值对（Map 不涉及原型链）。
