
## 分层列举

| **逻辑顺序** | **常用词（及格线）**       | **💯 高级替代（提分点）**                                                                 | **深度解析**                                    |
| -------- | ------------------ | -------------------------------------------------------------------------------- | ------------------------------------------- |
| **第一点**  | First, / Firstly,  | **First and foremost,**<br><br>**To begin with,**<br><br>**In the first place,** | “首先/首要的是”，比单一的 _First_ 显得更有气势和态度。           |
| **第二点**  | Second, / Besides, | **Furthermore,**<br><br>**Moreover,**<br><br>**In addition,**                    | “此外/而且”，用于递进。注意 _Moreover_ 后面往往接比第一点更重要的理由。 |
| **第三点**  | Finally,           | **Last but not least,**                                                          | “最后但同样重要的是”，完美解决最后一点论据显得单薄的问题。              |

## 弱转折与对比

| **功能**   | **常用词（及格线）**       | **💯 高级替代（提分点）**                          | **举个例子**                                                                       |
| -------- | ------------------ | ----------------------------------------- | ------------------------------------------------------------------------------ |
| **普通转折** | But / However,     | **Nevertheless,**<br><br>**Nonetheless,** | _The project is difficult. **Nevertheless,** it is worth trying._（然而，它依然值得一试。） |
| **正反对比** | On the other hand, | **Conversely,**<br><br>**By contrast,**   | “相反地/相比之下”。常用于对比两个群体（如：部分人支持；**相反地**，部分人反对）。                                   |
